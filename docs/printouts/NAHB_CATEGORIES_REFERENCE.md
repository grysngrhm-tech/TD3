**TD3 Construction Draw Management System**

## How to Use This Document

This reference maps the **NAHB (National Association of Home Builders) cost code taxonomy** to real builder category names from five different builders. Use this during team review sessions to:

1. **Verify mappings** - Confirm builder categories are mapped to the correct NAHB subcategory
2. **Identify gaps** - Find subcategories that may need to be added or removed
3. **Standardize terminology** - See how different builders name the same work items
4. **Mark changes** - Use the Review column checkboxes to flag items for discussion

## Important: Special Category Types

Some budget categories are **not draw categories** but are tracked for profit projection and loan structuring:

### Land / Lot Acquisition (Code 0210)

> **Not a draw category.** Land is typically funded by the builder as collateral for the construction loan. TD3 tracks this amount to calculate total project cost and estimate builder profit, but builders do not submit draw requests against this line item.

### Soft Costs (Code 0190)

The following categories are **estimated soft costs** used to project builder profit. They are not categories that builders draw against during construction:

| Category | Purpose |
|----------|---------|
| **Interest** | Estimated construction loan interest |
| **Financing Fee** | Loan origination and processing fees |
| **Realtor Fee** | Estimated sales commission |
| **Business Privilege Tax** | Local tax obligations |

These appear in builder budgets from: DT, LX, MR, CH
